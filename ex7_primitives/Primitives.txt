Примитивы и объекты

Примитивы(8) - это самый простой тип: boolean, chart, short, int, byte, long, float, double

Примитивы vs объекты:
У объекта - состояние и поведение (private String name)
У примитивов - только значение (private int price)

Переменная-объект - это по факту ссылка на этот объект в памяти
Переменная-примитив - просто значение, это не ссылка

Переменные работают похоже: при передаче в метод и при возврате из метода значения копируются

С помощью двойного равенства (==) примитивы сравниваются по значениям
А объекты - по ссылкам 

Переменные-объекты могут быть не заданы (null) 
Переменные-примитивы - всегда заданы
